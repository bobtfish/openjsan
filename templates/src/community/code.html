JSAN Git Repository

The code that runs JSAN is available in the JSAN Git Repository on GitHub. Checkout a copy and start hacking! Send your patches to the JSAN Development Mailing List or discuss them on irc.freenode.net in the #jsan channel. Once you've done that you're ready to get commit rights and you can contribute accordingly. 

git clone git://github.com/openjsan/openjsan.git
