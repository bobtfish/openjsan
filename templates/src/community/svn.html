JSAN Subversion Repository

The code that runs JSAN is available in the JSAN Subversion Repository. Checkout a copy and start hacking! Send your patches to the JSAN Development Mailing List or discuss them on irc.freenode.net in the #jsan channel. Once you've done that you're ready to get commit rights and you can contribute accordingly. 

svn co svn://master.openjsan.org/openjsan
